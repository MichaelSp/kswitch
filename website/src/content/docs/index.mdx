<section class="reveal" style="max-width:960px;margin:0 auto 4rem;text-align:center">
  <span class="section-eyebrow">Features</span>
  <p class="section-title">Everything operators need</p>

  <div class="features-grid">
    <div class="feature-card">
      <div class="feature-icon">⚡</div>
      <div class="feature-title">Instant fuzzy search</div>
      <div class="feature-desc">Search all contexts across every provider and kubeconfig in one keystroke. Hot-reload adds new files on the fly.</div>
    </div>
    <div class="feature-card">
      <div class="feature-icon">🖥</div>
      <div class="feature-title">Terminal isolation</div>
      <div class="feature-desc">Each terminal window targets a different cluster independently — kswitch never touches the original kubeconfig.</div>
    </div>
    <div class="feature-card">
      <div class="feature-icon">📜</div>
      <div class="feature-title">History navigation</div>
      <div class="feature-desc">Every context+namespace tuple is recorded. Jump back with <code>switch .</code> or step through with <code>switch -</code>.</div>
    </div>
    <div class="feature-card">
      <div class="feature-icon">🏷</div>
      <div class="feature-title">Context aliases</div>
      <div class="feature-desc">Give human-friendly names to cryptic generated context names without modifying the underlying kubeconfig.</div>
    </div>
    <div class="feature-card">
      <div class="feature-icon">🗄</div>
      <div class="feature-title">Search index cache</div>
      <div class="feature-desc">Local index for instant results across massive directories or slow remote stores like Vault.</div>
    </div>
    <div class="feature-card">
      <div class="feature-icon">🔌</div>
      <div class="feature-title">Extensible hooks</div>
      <div class="feature-desc">Run arbitrary executables before search — sync from Git, refresh tokens, rotate credentials.</div>
    </div>
  </div>
</section>
